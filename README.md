# Digital Finance - управление цифровыми финансами

Система управления цифровыми финансами. В приложении доступно:

	1. Регистрация пользователей
	2. Создание нескольких видов цифровых счетов
	3. Переводы между счетами и пользователями
	4. Пополнение счёта различными способами

## Используемые технологии

### Клиентская часть

Клиентская часть реализована на React JS с использование Bootstrap 5.

### Серверная часть

Серверная часть реализована на C# ASP.NET Core 6 по технологии WebAPI. 
Для документирования используется Swagger.
В качестве базы данных применяется PostgreSQL. Для работы с базой данных используется Dapper.
